## Organization Principles
- **src/**: Modular microservices (one dir per service).
- **Data Flow**: collector → streaming (Kafka) → detection → mitigation → monitoring.
- **Deployment**: Docker → K8s/Helm → Terraform (AWS EKS+MSK+Redis).
- **Security**: TLS certs, secrets mgmt, compliance module.

Data Flow: Traffic Source → Collector → Kafka → Detector → Mitigation + Prometheus/Grafana/Alerts

This structure supports scalable, production-grade DDoS protection.
